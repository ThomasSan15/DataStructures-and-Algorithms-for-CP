# DataStructures-and-Algorithms-for-CP

This repository contains my personal implementation of algorithms and data structures commonly used in Competitive Programming.

The main goal of this project is not only to build a reusable reference library, but also to deepen my understanding of the underlying concepts behind each algorithm and data structure. Whenever possible, implementations are written entirely by **myself** after studying the theory and design principles involved.

To support this learning process, I rely on books, academic resources, documentation, and other educational materials. Rather than simply copying existing code, I aim to understand how and why each technique works before implementing it.

This repository is a long-term learning project and will continue to grow as I explore new topics in algorithms, data structures, graph theory, dynamic programming, string processing, number theory, and other areas relevant to Competitive Programming.

*Objectives*
---
* Learn and implement algorithms from scratch.
* Study the theory behind each data structure and algorithm.
* Build a personal reference library for programming contests.
* Improve problem-solving and implementation skills.
* Document important ideas and techniques encountered during the learning process.

*Disclaimer*
---

The implementations in this repository are primarily intended for educational purposes. Code may be updated, optimized, or rewritten as my understanding of the topics improves.
